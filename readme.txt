# Chat Application with Task Assignment

This is a simple web-based chat application that allows users to communicate with each other and assign tasks within the chat. Users can send messages, and if they include a task assignment with `@username`, the task will be displayed in the task list without revealing the usernames.

![Chat Application ]

## Features

- Real-time chat functionality.
- Task assignment using `@username` mentions.
- Display of assigned tasks in the task list without revealing usernames.
- Minimalistic and user-friendly interface.

## Getting Started

These instructions will help you set up and run the chat application on your local machine for development and testing purposes.

### Prerequisites

You need a modern web browser to view and interact with the chat application. There are no additional software dependencies.
